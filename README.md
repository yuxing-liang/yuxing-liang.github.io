# Yuxing Liang Academic Website

This package contains the revised job-market version of the academic website.

## Deployment

Upload the **contents** of this folder directly to the root of the `yuxing-liang.github.io` repository. Do not upload this folder as a nested directory.

The repository root should contain:

```text
index.html
404.html
README.md
REVISION_REPORT.md
.nojekyll
css/
js/
assets/
CV.pdf
term-paper.pdf
```

## Included job-market updates

- Softer academic typography using Nunito Sans and Literata.

- Supervisor links restored.

## After upload

Go to GitHub: **Settings -> Pages -> Deploy from a branch -> main / root**.
Then wait a few minutes and hard refresh the site with `Ctrl + F5`.
