# Yuxing Liang - Academic Website

This is a static GitHub Pages website for Yuxing Liang.

## Deployment checklist

1. The repository name for a personal GitHub Pages site should be exactly `yuxing-liang.github.io`.
2. Publish from the `main` branch and the repository root.
3. `index.html` must be in the repository root, not inside a nested folder such as `yuxing-liang.github.io-main/`.
4. Upload or commit the contents of this folder, not the parent folder itself.
5. In GitHub, check **Settings > Pages** and confirm that the source is set to the root of the main branch.

If the site shows GitHub's generic 404 / "Not Found" page, the most common causes are: the repository name is wrong, GitHub Pages is publishing from the wrong branch/folder, or `index.html` is not located at the publishing root.
